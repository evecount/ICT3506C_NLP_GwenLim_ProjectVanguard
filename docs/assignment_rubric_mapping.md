# Module ICT3506C Natural Language Processing — Rubric & Competency Mapping

## Objective
This portfolio submission serves as a verifiable demonstration of competency to challenge and clear the ICT3506C Natural Language Processing module. 

## 1. Core Technical Competencies Showcased in Project Vanguard

Project Vanguard has been architected specifically to map directly to the required competencies outlined in the challenge assessment:

* **NLP Fundamentals & Text Processing (Tokenization, Cleaning, Normalization):**
  - Handled by the ephemeral ingestion gateway, where raw chaotic chat telemetry is normalized and stripped of PII before tokenization.
* **Representations (Word Embeddings):**
  - Text strings are transformed directly into high-dimensional floating-point coordinate arrays ($\mathbb{R}^d$) forming the local vector manifold.
* **Analysis (Syntax and Semantic Analysis):**
  - Moving beyond basic sentiment, Vanguard maps category graphs to track *hyper-accelerated semantic escalation gradients* and structural intent.
* **Applications (Text Classification and Dialogue Systems):**
  - *Classification*: Identifying Anomalous Risk Profiles vs Baseline Users.
  - *Dialogue Systems*: Modeling sequential multi-user chat dynamics rather than isolated sentence parsing.
* **Implementation (Hands-on Model Implementation & Libraries):**
  - Utilizing Python for the baseline linear algebra/embedding generation (MVP) and the Wolfram Engine for high-level Symbolic/Topological Data Analysis.

## 2. Submission Artifacts

To satisfy the portfolio requirements, the following artifacts are provided within this repository:
* **Code Repositories (`mvp/`):** Clean, documented Python code showing the NLP processing pipeline and vector distance mechanics.
